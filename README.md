# photoshop

# 화면크기 조절
- alt + 마우스 휠 위아래로 조절
- ctrl + 0 => 화면크기에 맞춤
- ctrl + 1 => 100% 크기로 맞춤

# 레이어 복사
- ctrl + j
- alt + 레이어를 위로 이동 => 복사된 레이어가 위에 위치
- alt + 레이어를 아래로 이동 => 복사된 레이어가 아레에 위치
  
# 이미지 크기조절
- ctrl + T
- 조절시 alt + 조절 => 기준을 가운데로 변경

# group 만들기
- ctrl + G => 만들기
- ctrl + shift + G => 그룹풀기

# trim 
- 사진 주위에 투명한 부분 정리하여 사진크기에 맞추기 (image > trim)

# 화면이동
- space bar + 화면을 드래그

# 선택영역 없에기
- ctrl + D
